### SUMMARY

some DIY application launcher program for windows written in c++.

### BUILDING COMMAND (for example)

```
c++ foo.cxx -o foo.exe -static
```

### BUILDING ENVIRONMENT

#### ENVIRONMENT

```
MSYS2
```

#### ADDITIONAL PACKAGES

```
base-devel
mingw-w64-x86_64-toolchain
```
